```python
# 关键差异
# Compactor 辊径：25, 30, 50, 75, 100 mm
# Granulation 辊径：120, 250 mm
# 合并后覆盖：25–250 mm（7个尺寸）→ 可外推到电池压延机300–600mm

# Compactor SCF范围：0.52–3.24 kN/cm（比Granulation低很多）

# Compactor独有特征：vfs_hfs_ratio（预压控制，脆性材料最敏感）
# Compactor缺失：peak_pressure_MPa, densification_factor, nip_angle_deg
# → 用Stage-1 XGBoost从(scf, gap, roll_diameter)预测补算
```

**合并后 Leave-One-Scale-Out CV 结果（从今天代码得到）：**
- 合并后对各辊径的R²比单独Granulation更稳定
- 25mm、30mm小机器只有合并数据才能预测（Granulation根本没这个尺寸）
